# Project Overview
This is a full-stack application using a React/TypeScript frontend and a Python/FastAPI backend. 

# Architecture & Directories
- `/frontend`: Contains the React/Vite web application.
- `/backend`: Contains the Python 3 FastAPI server.

# Global Rules
- IMPORTANT: When working on UI components, refer to the rules in `frontend/CLAUDE.md`.
- IMPORTANT: When working on API routes or database models, refer to the rules in `backend/CLAUDE.md`.
- Never commit secrets or API keys.
